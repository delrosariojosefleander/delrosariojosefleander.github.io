Trying Javascript for the First Time

31 Aug 2023

JavaScript is a programming language that I’ve been studying and using for about a week now. Although it shares some features with Java, a language I am more familiar with, there are many distinct differences that initially confused me while studying JavaScript. One of these differences relates to how variables are initialized in JavaScript. In Java, to initialize a variable, you must declare its data type. For example, to declare a variable of type ‘int’, you must specify the data type before the variable’s name. In JavaScript, this is unnecessary, as all variables can be initialized without declaring a data type. Instead, what distinguishes variables in JavaScript is their mutability and scope. For instance, a variable declared with ‘var’ has global scope and mutability, while variables declared as ‘let’ and ‘const’ have limited scope, with ‘const’ having no mutability.

However, JavaScript is much more streamlined than Java. Without the limitations imposed by strict rules like in Java, writing in JavaScript is not only simpler but also more versatile. For example, writing methods in JavaScript can have multiple types of returns of any data type, while Java only allows for the return of one data type. This allows for more flexible methods and streamlines the writing process. Instead of writing several different methods to handle various data types, it is possible to write a single method to suit your needs.

Java and JavaScript have their own separate rules and requirements; however, the syntax appears so similar that at a glance, they seem extremely alike. They even share the word “Java” in their names. However, upon further study, these two programming languages couldn’t be more different. Initially, overcoming these differences seemed difficult and rather daunting at times. But as I studied JavaScript more, I began to prefer the streamlined nature of JavaScript over the rigidity of Java. My experience learning JavaScript was much like trying a strange and seemingly off-putting dish for the first time and enjoying it. The moral of the story: don’t be afraid to try new things. In the future, I plan to deepen my understanding of the JavaScript language.
